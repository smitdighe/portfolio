* 🎯 Scroll Reveal Animations

## ⚙️ Run Locally

1. Clone the repository

```bash
git clone https://github.com/smitdighe/portfolio.git
```

2. Navigate into the folder

```bash
cd portfolio
```

3. Run in browser

```bash
open index.html
```
